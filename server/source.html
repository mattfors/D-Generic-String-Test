Installing Required Libraries (libserial, FFTW, Arduino)
    Automatic: librariesinstall.sh
    Installs libraries required to compile program.
        To run:
             (1)Open terminal and navigate to directory
             (2)Enter: "bash librariesinstall.sh" without quotes
             (3)Enter password when prompted
    Manual
        Enter: "sudo apt-get install libserial0 libserial-dev fftw2 fftw-dev avr-libc gcc-avr" without quotes.

Port Detection
    Automatic: portdetect.sh
    Compares the contents of /dev/ directroy to determine the location of added device. Generates the config file required by program.
        To run:
             (1)Open terminal and navigate to directory
             (2)Enter: "bash portdetect.sh" without quotes
             (3)Follow instructions given by script
    Manual
        You can find the location of an Arduino board by looking in the /dev/ directory or using the Arduino IDE. In the Arduino IDE go to tools>Serial Port, a check mark should be next to the device. If you already know the location you can manual add it to the config file
            (1)Open "config" in a text editor
            (2)Add the location to the first line of the file
            (ex.) "/dev/ttyACM0" without quotes

To Compile:
        (1)Open terminal and navigate to directory of Makefile
        (2)Enter: "make" without quotes
